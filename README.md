# codetree-TILs

CodeTree와 알고리즘 문제 풀이를 자동으로 기록하는 학습 저장소입니다.

이 저장소는 문제 풀이 파일이 날짜와 문제명 기준으로 누적되는 아카이브 성격의 repo입니다. 포트폴리오 대표 프로젝트라기보다는 꾸준한 문제 풀이 기록과 학습량을 보여주는 보조 자료로 활용합니다.

## 저장소 목적

- CodeTree 문제 풀이 기록 관리
- Java/Python 기반 알고리즘 풀이 누적
- 날짜별 학습 흐름 확인
- 구현력과 문제 해결 루틴 유지

## 활용 방법

문제별 폴더 안에 풀이 코드와 자동 생성된 README가 함께 저장됩니다.

```text
날짜/문제명/
├── README.md
└── solution.java 또는 solution.py
```

## 포트폴리오 메모

이 저장소는 실행형 서비스 프로젝트가 아니라 알고리즘 풀이 아카이브입니다. GitHub 프로필에서는 주요 백엔드 프로젝트 아래에 `Algorithm / TIL` 성격으로 배치하는 것이 좋습니다.
